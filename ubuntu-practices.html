1 安装

1.1 硬盘安装

使用EFI引导模式安装，由于EFI能够自动识别磁盘根目录下的EFI文件夹中的引导文件，所以只需要把ISO文件解压到U盘即可，主要U盘需要先格式化为FAT32位的，这里有一个安装教程，安装过程中最重要的一点是要建立EFI引导分区，否则不能引导成功。

安装过程中如可以下载更新的话推荐直接下载更新，因为有次安装的之后更新系统导致一些软件不能正确安装。此问题待有时间深究一下。

1.2 虚拟机安装

使用 VMware Workstation 安装。有必要的话安装 vmware-tools。

3.2 ibus-rime中州韵

目前（2015.02.10）通过ppa安装后语言栏不能出现图标，所以建议使用手动安装，参看 ibus-rime安装说明 中 Manual Installation小节。考虑到Google被封锁，整理安装过程如下： 

cd ~/projects
wget -P ~/Downloads http://dl.bintray.com/lotem/rime/brise-0.35.tar.gz
wget -P ~/Downloads http://dl.bintray.com/lotem/rime/librime-1.2.tar.gz
wget -P ~/Downloads http://dl.bintray.com/lotem/rime/ibus-rime-1.2.tar.gz
sudo apt-get install cmake libboost-all-dev libgoogle-glog-dev libgtest-dev libibus-1.0-dev libnotify-dev libkyotocabinet-dev kyotocabinet-utils libmarisa-dev libopencc-dev libyaml-cpp-dev
tar xzf ~/Downloads/librime-*.tar.gz
tar xzf ~/Downloads/brise-*.tar.gz
tar xzf ~/Downloads/ibus-rime-*.tar.gz
cd ibus-rime
# do this as normal user
./install.sh

5 安装 chrome

5.1 标签页乱码

打开/etc/fonts/conf.d/49-sansserif.conf这个文件： 

sudo gedit /etc/fonts/conf.d/49-sansserif.conf

然后修改倒数第四行的字体为WenQuanYi Zen Hei，如下: 

<?xml version="1.0"?>
<!DOCTYPE fontconfig SYSTEM "fonts.dtd">
<fontconfig>
  <!--
      If the font still has no generic name, add sans-serif
  -->
  <match target="pattern">
    <test qual="all" name="family" compare="not_eq">
      <string>sans-serif</string>
    </test>
    <test qual="all" name="family" compare="not_eq">
      <string>serif</string>
    </test>
    <test qual="all" name="family" compare="not_eq">
      <string>monospace</string>
    </test>
    <edit name="family" mode="append_last">
      <string>WenQuanYi Zen Hei</string>
    </edit>
  </match>
</fontconfig>

之后flash中文和chromium中文都可以正常显示了。

7 安装 emacs

# 或者 emacs 开发版本
git clone -b master git://git.sv.gnu.org/emacs.git
cd emacs
./autogen.sh

# 配置
