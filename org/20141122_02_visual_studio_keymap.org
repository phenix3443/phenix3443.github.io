注释选定块：Control+K,Control+C

查找和替换：Control+F\Control+H

查找所有引用：Shift+F12
删除整行：Control+L
反撤销：Control+Y(如果是mac用户+fn)
折叠代码定义：CTRL + M, CTRL + O
展开代码定义：CTRL + M, CTRL + L
