#+TITLE:使用cygwin编译 GNU Global

* 安装 cygwin
1. 选择安装位置、下载位置、源（163），其余选项默认。
2. 不要贪图省事选全部安装，这样在安装结束的时候会一直提示“incomplete install”，根据需要安装。
3. 如果下载时出事进度条卡顿，重新安装的除了将 install 换位 reinstall，不要更改其他设置。
4. 如果点击桌面图标，但是提示正在查找 mintty，先通过安装程序安装 mintty，然后更改桌面快捷图片属性中 url，将其中的 mintty 改为 mintty.exe。
