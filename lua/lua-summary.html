9 编程范式

9.1 OOP

使用元表（metatable）实现面向对象： 

local Human = {
	name="noname"
}

function Human:new(o)
	local o = o or {}
	setmetatable(o,self)		-- 注意
	self.__index = self
	return o
end

function Human:say()
	print("hello, i am", self.name)
end

-- 注意如果类函数定义是冒号，实例使用方法的时候也要是冒号
local people = Human:new()
people:say()					--  hello, i am	noname

注意，Human:new函数中两处用的都是self，而不是Human，这里的作用是防止Human被赋值给其他变量之后该类不能使用。 

local People = Human
local p2 = People:new()
p2:say() -- hello, i am	noname

9.2 继承

需要仔细思考：

  • 元表的作用。
  • __index = self
  • class:new在定义class实例和子类中self和元表的不同用处。（关键） 
local Human = {
	name="noname"
}

function Human:new(o)
	local o = o or {}
	setmetatable(o,self)		-- 注意，此次用到self就是实现继承的关键
	self.__index = self
	return o
end

local Man = Human:new()
Man.gender = "male"

local Woman = Human:new()
Woman.gender = "famale"

function Human:get_gender()
	if self.gender then
		print("i am", self.gender)
	end
end

local m = Man:new()
local w = Woman:new()
m:get_gender()					-- i am	male
w:get_gender()					-- i am	famale
