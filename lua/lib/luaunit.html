1 概述

LuaUnit 是一个流行的 Lua 单元测试框架，具有典型的 xUnit 库（比如 Python 的 unittest，Junit 等）接口。它支持多种输出格式（文本，TAP，JUnit，…）来直接使用或与持续集成平台（Jenkins，Maven，…）一起使用。

LuaUnit 提供了丰富的 assert 机制，并且在输出结果上也做的不错，比如，对比 table 的时候可以输出不一致的项。

LuaUnit 还可以从命令行设置要执行的单元测试项目、输出格式，还有其他各种各样的配置。

2 安装

建议使用 luarocks 安装： 

luarocks install luaunit

3 使用

LuaUnit 扫描所有以 test 或 Test 开头的变量。如果它们是函数，或者它们是包含以 test 或 Test 开始的函数的表，则它们作为测试套件的一部分运行（这点和 Python unittest 类似）。

注意这些变量要放在全局空间。

3.1 命令行

lua <your_test_suite.lua> [options] [testname1] [testname2]

还可以通过使用–pattern 参数匹配 testsuit 和 testcase 的名字，来指定运行示例。
