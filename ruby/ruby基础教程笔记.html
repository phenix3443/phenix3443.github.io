1 第一章 Ruby 初探

  • 单引号中的特殊字符不经过转义。
  • p 方法可以区分输出的是数字还是字符串。
  • 使用 # 作为注释符号。 =begin 和 =end 括起来的部分也是注释。
  • 使用 inlucde 导入库。
  • 条件判断： if then else end
  • 循环：
    • while do end
    • 循环次数.times do

2 第二章 便利的对象

  • 数组
    • 形式： 
      lang = ["c", "c++", "ruby"]
    • 可以用来保存对象，支持不同类型对象的混合保存，下标从 0 开始。
    • 方法： 
      print(lang.size)
    • 遍历： 
      lang.each do |e|
  print(e)
end
  • 散列
    • 形式： 
      person = {name: "ruby", :age => 18}
    • 存储键值对
    • 散列中使用符号做 key 要比字符串更有效率。
    • 遍历： 
      person.each do |key, value|
  print(key,value)
end
  • 正则表达式
    • 形式： 模式 
      /Ruby/ =~ "ruby"
    • 使用 =~ 来匹配字符串，成功返回匹配部分的位置，失败返回 nil。

3 第三章 创建命令

  • 内置数组 ARGV 表示命令行传递的参数。0 表示第一个参数，而不是脚本文件。
  • 遍历文件： 
    file = File.open('test.txt')
file.each_line do |line|
  puts(line)
end
  • 定义方法： 
    def say_hello
  puts("hello")
end
  • 使用 require 引用其他文件。
