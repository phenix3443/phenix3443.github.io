1 安装

1.1 硬盘安装

使用EFI引导模式安装，由于EFI能够自动识别磁盘根目录下的EFI文件夹中的引导文件，所以只需要把ISO文件解压到U盘即可，主要U盘需要先格式化为FAT32位的，这里有一个安装教程，安装过程中最重要的一点是要建立EFI引导分区，否则不能引导成功。

安装过程中如可以下载更新的话推荐直接下载更新，因为有次安装的之后更新系统导致一些软件不能正确安装。此问题待有时间深究一下。

1.2 虚拟机安装

使用 VMware Workstation 安装。有必要的话安装 vmware-tools。

5 安装 goagent

goagent on github

Linux 下需要安装一些依赖组件：

  • gevent 1.0(提升多线程性能，强烈建议安装)
  • greenlet (gevent的依赖，一般安装gevent会自动安装)
  • python-vte(基于GTK的简单GUI所需)
  • python-openssl 0.13(生成证书所需，强烈建议安装，如删除了goagent自动证书则必须安装)
  • pycrypto(RC4加密所需，建议安装)
  • python-appindicator(Unity桌面下的托盘组件，其他桌面不必安装)

注意插件安装的先后顺序最好不要改,因为排列在后面的插件需要依赖前面的插件才能运行起来.并且自己酌情安装,比如你不是使用的ubuntu那么和Unity桌面相关的玩意儿,就可以不需要了,把它从上述命令里剔除掉就可以.

如果greenlet版本低于0.4.0会导致gevent装不上，但最新的ubuntu更新下来的greenlet就已经是0.4.2了,所以不用太担心.如果是老版本的那么就需要查询下更新greenlet了. 

# 安装第三方软件
sudo apt-get install python-dev python-greenlet python-gevent python-vte python-openssl python-crypto python-appindicator
cd ~/projects/
git clone git@github.com:goagent/goagent.git
cd goagent
# 上传服务器代码
python server/upload.py
# 修改 appid
gedit local/proxy.ini

使用 root 账户身份去启动goagent.会发现有新的证书被生成了.

Firefox可以直接在浏览器中导入证书，但是 chrome 证书机制不同，需要使用公用nssdb库，需要做以下配置。 

# 安装证书
sudo apt-get install libnss3-tools
# 查看现有证书目录
certutil -d sql:$HOME/.pki/nssdb -L
# 如果出现以下错误：
certutil: function failed: security library: bad database.
# 说明证书损坏，需要重建：
rm -fr ~/.pki/
mkdir -p ~/.pki/nssdb
chmod 700 ~/.pki/nssdb
certutil -d sql:$HOME/.pki/nssdb -N # 之后输入密码
# 导入 goagent 证书
certutil -d sql:$HOME/.pki/nssdb -A -t "C,," -n GoAgent -i "~/projects/goagent/local/CA.crt"
# 开机自动
sudo python ~/projects/goagent/local/addto-startup.py

到这里,恭喜,已经全部完成了.可以尽情的使用goagent了. 注意： chrome一切都配置好了，打开谷歌等网站就提示：Your connection is not private,可能是之前chrome导入的goagent证书有问题，可以尝试在 https/ssl里先删掉老的goagent证书，然后重新导入一下。

7 安装 chrome

7.1 标签页乱码

打开/etc/fonts/conf.d/49-sansserif.conf这个文件： 

sudo gedit /etc/fonts/conf.d/49-sansserif.conf

然后修改倒数第四行的字体为WenQuanYi Zen Hei，如下: 

<?xml version="1.0"?>
<!DOCTYPE fontconfig SYSTEM "fonts.dtd">
<fontconfig>
  <!--
      If the font still has no generic name, add sans-serif
  -->
  <match target="pattern">
    <test qual="all" name="family" compare="not_eq">
      <string>sans-serif</string>
    </test>
    <test qual="all" name="family" compare="not_eq">
      <string>serif</string>
    </test>
    <test qual="all" name="family" compare="not_eq">
      <string>monospace</string>
    </test>
    <edit name="family" mode="append_last">
      <string>WenQuanYi Zen Hei</string>
    </edit>
  </match>
</fontconfig>

之后flash中文和chromium中文都可以正常显示了。
