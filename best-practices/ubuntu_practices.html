1 安装

1.1 硬盘安装

使用 EFI 引导模式安装，由于 EFI 能够自动识别磁盘根目录下的 EFI 文件夹中的引导文件，所以只需要把 ISO 文件解压到 U 盘即可，主要 U 盘需要先格式化为 FAT32 位的，这里有一个安装教程，安装过程中最重要的一点是要建立 EFI 引导分区，否则不能引导成功。

安装过程中如可以下载更新的话推荐直接下载更新，因为有次安装的之后更新系统导致一些软件不能正确安装。此问题待有时间深究一下。

1.2 虚拟机安装

使用 VMware Workstation 安装。有必要的话安装 vmware-tools。

7.12 KDE






7.12.1 安装





KDE Plasma desktop latest packages are available under kubuntu-ci/stable PPA. Use the following commands to add this PPA to your system and install plasma-desktop package.







sudo apt-get install kubuntu-desktop

sudo add-apt-repository ppa:kubuntu-ci/stable
sudo apt-get update
sudo apt-get install kubuntu-desktop












7.12.2 卸载





KDE plasma is a good desktop environment, but you still don’t like it, then you can revert all changes with following command.







sudo apt-get install ppa-purge
sudo ppa-purge ppa:kubuntu-ci/stable

7.16 PlayOnLinux





以安装 安装 RTX2015 为例：







wget -q "http://deb.playonlinux.com/public.gpg" -O- | sudo apt-key add -
sudo wget http://deb.playonlinux.com/playonlinux_trusty.list -O /etc/apt/sources.list.d/playonlinux.list
sudo apt-get update
sudo apt-get install playonlinux





    

  • 打开 playonlinux 界面

    • 

  • 点击工具, 选择管理 wine 版本, 然后选择 (amd64), 当然 32 位选择 X86, 选择一个可以用的,装好就行。

    • 

  • 点击配置

    • 

  • 进入配置页面

    • 

  • 选择新建, 点击下一步, 创建一个 64 位的虚拟盘

    • 

  • 选择刚才安装的 Wine，或者没人 system

    • 

  • 点击下一步, 输入虚拟盘名称, 然后完成了, 我这里是 rtx。

    • 

  • 安装相应组件，点击虚拟盘, 选择安装内容, 安装相应组件. 下面是需要的组件列表, 一个都不能少哦：msxml3 msxml6 ie8 vcrun6 vcrun2005 riched20 riched30

    • 

  • 安装 RTX，回到主界面, 选择安装一个程序

    • 

  • 点击安装未在列表的程序

    • 

  • 点击编辑或更新现有的应用程序

    • 

  • 点击显示虚拟盘, 选择刚才新建的虚拟盘 rtx。

    • 

  • 什么不用勾选, 点击下一步

    • 

  • 点击 64 位安装, 点击下一步

    • 

  • 点击浏览, 选择 RTX 安装包安装就可以了, 这里可以下载安装包

    • 





安装好以后不管怎么弄界面都是乱码，等待以后继续折腾。
