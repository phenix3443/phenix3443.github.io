Org Mode 是 Emacs 的一个杀手级的应用，可以用来做笔记、维护 TODO List、做项目管理等等，结合 org2blog mode 还可以直接发布文章到博客。

1 关于本文

这篇文章假设读者对 Org Mode 有一定的了解，对一些概念，比如 agenda、capture mode，不会一无所知。如果你还是新手，建议先去官网1阅读 guide2, 更多细节的知识则参考 manual3。

我也是刚刚开始使用 Org Mode，本文主要记录了我的相关配置，有些配置是默认配置，我也明确的写出来，这样重要的配置很明确，及时以后版本更改，这些配置也不会改变，使用过程中也会不断的修改，不定期更新。

2 许可证

这篇文章的 html 和 org 形式都基于 GPL V3 发布。

3 更新历史

本文现在的版本是 callorg-mode-doc-version，通过 org-mode version callorg-mode-version 进行发布，这篇文章的源码在个人 github 上，更新时间不定。

  • 使用 C-c ' 可以源码在代码段（#+BEGINSRC）中正常缩进。[fn:5]

    • 

  • 插入 shell 脚本的时候：

      

    • #+BEGINSRC shell 在发布 html 时候终端，出现错误：“(wrong-type-argument processp nil)”。

      • 

    • #+BEGINSRC shell-script 再输入 script 最后一个字幕t的时候会有卡顿，提示“Indentation setup for shell type bash”。

      • 

    • #+BEGINSRC bash 不能进入 shell 环境编辑。

      • 

    • #+BEGINSRC sh 可以正常使用。
